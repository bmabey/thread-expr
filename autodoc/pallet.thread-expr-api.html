Overview


Macros that can be used in an expression thread




Public Variables and Functions



  

  

  
apply->

  macro

  
Usage: (apply-> request f & args)


  
Apply in a threaded expression.
e.g.
   (-> 1
     (apply-> + [1 2 3]))
=> 7

  


  

  

  
apply-map->

  macro

  
Usage: (apply-map-> request f & args)


  
Apply in a threaded expression.
e.g.
   (-> :a
     (apply-map-> hash-map 1 {:b 2}))
=> {:a 1 :b 2}

  


  

  

  
arg->

  macro

  
Usage: (arg-> arg [sym] & body)


  
Lexically assign the threaded argument to the specified symbol.

(-> 1
  (arg-> [x] (+ x)))

=> 2

  


  

  

  
binding->

  macro

  
Usage: (binding-> arg bindings & body)


  
A `binding` form that can appear in a request thread.
eg.
   (def *a* 0)
   (-> 1
     (binding-> [*a* 1]
       (+ a)))
=> 2

  


  

  

  
for->

  macro

  
Usage: (for-> arg seq-exprs & body)

eg.

     (for-> [x [1 2 3]]
       (+ x)))
